隐私政策

AI投顾助手（com.qingqiuma.aiassistant）
生效日期：2026年2月22日  |  最近更新：2026年2月22日

引言

AI投顾助手（以下简称"本应用"）由 qingqiuma 开发。我们非常重视用户的隐私保护，本隐私政策旨在向您说明我们如何收集、使用、存储和保护您的个人信息。请您在使用本应用前仔细阅读本隐私政策。

使用本应用即表示您同意本隐私政策的内容。如您不同意本政策的任何内容，请停止使用本应用。

一、信息收集

1.1 本应用不主动收集的信息

本应用 不收集、不上传、不共享 以下信息：

  • 用户的真实姓名、身份证号、手机号码等个人身份信息
  • 用户的地理位置信息
  • 用户的通讯录、短信、通话记录
  • 用户的相册、照片、文件
  • 设备的 IMEI、MAC 地址等唯一设备标识

1.2 本应用本地存储的信息

以下信息仅存储在您的设备本地，不会上传至任何服务器：

数据类型用途存储位置
自选股列表展示您关注的股票行情设备本地
补仓计算历史方便您查看历史计算记录设备本地
AI 服务配置存储您自行配置的 AI API Key设备本地
通知偏好设置记录您的推送开关和提醒时间设备本地
深色模式设置记录您的显示模式偏好设备本地
应用锁设置存储您设置的应用锁密码（加密存储）设备本地

二、网络请求

本应用需要访问互联网以获取以下公开数据：

数据类型数据来源用途
实时股票行情腾讯财经、东方财富（公开 API）展示股票价格、涨跌等行情数据
财经新闻新浪财经、同花顺、雪球、东方财富（公开 API）展示财经资讯
AI 分析DeepSeek、通义千问（用户自行配置 API Key）提供 AI 辅助分析功能
重要说明：以上所有网络请求仅用于获取公开的市场数据和新闻资讯，不包含任何用户个人信息的上传。AI 服务的 API Key 由用户自行从第三方平台获取并配置，本应用不提供也不存储任何预设的 API Key。

四、数据安全

我们采取以下措施保护您的数据安全：

  • 本地存储：所有用户数据仅存储在设备本地，不上传至任何云端服务器。
  • 无账户体系：本应用无需注册登录，不收集用户账户信息。
  • API Key 保护：您配置的 AI API Key 存储在设备本地应用沙箱中，仅在发起 AI 分析请求时使用。
  • 应用锁：支持密码和生物认证（指纹/面容）保护应用访问，防止他人查看您的数据。

五、第三方服务

本应用使用以下第三方公开服务获取市场数据，这些服务有各自的隐私政策：

  • 腾讯财经 — 股票行情数据
  • 东方财富 — 股票列表、公告、新闻
  • 新浪财经 — 财经新闻、视频
  • 同花顺 — 快讯新闻
  • 雪球 — 社区热门内容
  • DeepSeek / 通义千问 — AI 分析服务（需用户自行配置）

以上第三方服务的数据使用和隐私保护受其各自隐私政策约束，建议您在使用前了解相关平台的隐私政策。

六、未成年人保护

本应用面向成年用户。我们不会有意收集未满 18 周岁的未成年人的个人信息。如果您是未成年人，请在监护人的指导下使用本应用。
